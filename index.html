API Cantantes

Esta API permite obtener información de los cantantes de la base de datos.

Para obtener la información de un cantante, se debe hacer una petición GET a la siguiente URL:

https://extinct-erin-salmon.cyclic.app/cantantes/:id

Donde :id es el identificador del cantante.

Por ejemplo, para obtener la información del cantante con identificador 1, se debe hacer una petición GET a la siguiente URL:

https://extinct-erin-salmon.cyclic.app/cantantes/1
